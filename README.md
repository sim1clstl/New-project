# Mom Day Ultra Pro Max

A Vite + React Mother's Day site with Framer Motion, GSAP scroll animation,
real photos, and intentionally excessive mom propaganda.

## Run Locally

```powershell
npm install
npm run dev
```

## Deploy To Vercel

Import this folder as a Vercel project.

Use these settings:

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
